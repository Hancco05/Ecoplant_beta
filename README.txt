# Ecoplant Project

Este es el proyecto Ecoplant creado con Laravel 10.

## Estructura del Proyecto

- **Página de Inicio**
  - Mensaje de bienvenida
  - Menú de navegación con enlaces a Home, Blog y Contacto

- **Blog**
  - Lista de noticias
  - Detalles de cada noticia
  - Funciones de edición para administradores

- **Formulario de Contacto**
  - Formulario para enviar información de contacto
  - Integración con servicio de correo para respuestas

## Requisitos

- PHP 8.1 o superior
- Composer
- Node.js y npm
- MySQL
- Laravel 10
